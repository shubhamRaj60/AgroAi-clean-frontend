About AgroAI

AgroAI is a smart agriculture system designed to enhance decision-making for farmers using modern technology. It combines real-time soil monitoring, crop prediction, feedback collection, and data visualization into a single web-based platform.

  • 🌱 Predict the best crop based on soil and environmental data
  • 📊 Visualize sensor data (TDS, moisture, temperature)
  • 💬 Submit and review user feedback
  • 🔌 Powered by ESP32, Firebase, and Node.js

Tech Stack

HTML5 CSS3 JavaScript Bootstrap 5 Firebase Realtime DB Node.js (Express) Python (ML Model) ESP32 + Soil Sensors Chart.js GitHub Pages

Why AgroAI?

🎯 Targeted Crop Suggestions

Improve yields by planting crops suited to your specific soil and weather conditions.

📉 Data-Driven Decisions

Use live sensor data (moisture, temp, TDS) to make smarter irrigation and planting decisions.

💬 Farmer Feedback Loop

Collect suggestions, problems, and ratings directly from users to improve the system.
